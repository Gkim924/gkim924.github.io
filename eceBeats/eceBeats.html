FPGA Music Player - xECEBEATS

Go Back

This project implements a remote-controlled music player on the FPGA that interfaces with a VGA monitor to display the song information and cover art. The FPGA fabric serves as the main control unit of the music player, and it interfaces with various components on the DE-115 board to process data such as the:

  • Music samples
  • Cover art image
  • Title image
  • Remote control signals
  • Status icons (Play, Pause, Playback Speed)

This project is entirely hardware based for improved performance and does not rely on a SoC (NIOS II).
